# Philodendron Knowledge Base

เอกสารชุดนี้เป็นคู่มือและบันทึกหลักฐานสำหรับสกุล Philodendron รุ่นแรกของ Philodendron Lab

## ขอบเขต

- taxonomy backbone: Kew POWO / WCVP
- taxonomy catalog: ใช้ snapshot จำนวนชนิดที่ยอมรับและบันทึกวันที่ตรวจสอบ
- monograph รุ่นแรก: Pink Princess และ Violin variegated
- tissue culture: เริ่มจาก nodal culture และแยก `Verified`, `Adapted`, `Experimental`, `Pending review`

## กฎการอ่านคู่มือ

1. ชื่อผู้ขายไม่ใช่หลักฐานยืนยัน species หรือ cultivar
2. cultivar/trade-name ต้องผูกกับ parent species
3. สูตรจากงานวิจัยใช้ได้เฉพาะบริบทของงานนั้นจนกว่าจะ validate กับระบบของเรา
4. Violin variegated ยังไม่มี protocol ตรงพันธุ์ในฐานข้อมูลนี้ จึงเริ่มทุกคำแนะนำเป็น Experimental/Adapted
5. การทำต้นที่สองหรือต้นที่สามต้องสร้าง Plant Record และ Experiment Lot ใหม่ ไม่รวมผลข้ามต้นอัตโนมัติ
