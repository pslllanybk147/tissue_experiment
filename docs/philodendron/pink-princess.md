# Philodendron erubescens ‘Pink Princess’

สถานะคู่มือ: `Adapted` สำหรับ workflow ห้องทดลองนี้

## หลักฐานตรงพันธุ์

งานปี 2023 รายงานว่า MS + BAP 1.0 mg/L ให้ผลดีต่อการเพิ่มยอดในระบบที่ศึกษา และ solid MS + IBA 3.0 mg/L ให้ผลดีต่อการออกราก งานปี 2025 ศึกษา temporary immersion และ genetic fidelity เพิ่มเติม รายละเอียดเหล่านี้เป็นหลักฐานของระบบวิจัยใน paper ไม่ใช่การรับรองผลกับอุปกรณ์หรือสารของผู้ใช้โดยอัตโนมัติ

## วิธีเริ่มสำหรับผู้ใช้ใหม่

1. สร้าง Plant Record และถ่าย baseline ทั้งต้น ใบ ยอด ข้อ และรากที่เห็น
2. กักต้นและตรวจสุขภาพก่อนตัด
3. เลือก nodal explant ที่มีตาข้างชัด เก็บข้อสำรองไว้บนต้นแม่
4. ทำ blank/control ก่อนใช้ต้นราคาแพง
5. สร้าง Experiment Lot แยกจากต้นอื่น
6. ใช้ Guided Protocol ในเว็บทีละขั้น และบันทึกภาพ/ค่า/note ทุกจุดสำคัญ

## 18 ขั้นในระบบ

รับต้น → กักต้น → ยืนยันชนิด → เลือก explant → ตัด → เตรียมพื้นที่ → เตรียมอาหาร → ฟอก → ลงอาหาร → ตรวจ contamination → เพิ่มยอด → ออกราก → ปรับสภาพ → ติดตามลายด่าง → troubleshooting → review หลักฐาน → export record → ปิด Lot

## ข้อควรระวัง

- อย่านำเวลาหรือความเข้มข้นฟอกของกุหลาบหรือ Philodendron ชนิดอื่นมาแสดงเป็น Verified
- อย่าใช้สีใบเพียงอย่างเดียวเป็นหลักฐาน genetic fidelity
- เมื่อไม่ผ่าน ให้เปลี่ยนทีละตัวแปรและบันทึกเป็น Experimental
- หากพบรา เมือก หรือเนื้อเยื่อตายต่อเนื่อง ให้หยุด/แยกภาชนะตาม SOP ความปลอดภัย
